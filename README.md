# AAAC++学习助手

AAAC++学习助手是一个基于Qt开发的学习应用，用于学习 C++ 面向对象编程中的类相关知识。本应用以章节和小节组织课程内容，包含知识讲解、代码示例、练习题和学习进度记录等功能。

## 功能

- 通过左侧导航浏览课程章节和知识小节
- 查看文本讲解与 C++ 代码示例
- 通过练习题进行自测
- 标记小节完成状态
- 在学习总览中查看章节完成情况
- 将学习进度保存到 `progress.json`

## 项目结构

```text
.
+-- main.py          # 应用入口和主要界面逻辑
+-- progress.json    # 本地学习进度数据（首次运行后自动生成）
+-- README.md
```

## 运行环境

本项目使用Python书写，需要安装PySide6：

```bash
pip install PySide6
```

## 运行方式

在项目所在目录执行：

```bash
python main.py
```

应用启动后会打开 `AAAC++学习助手` 桌面窗口。学习完成状态会自动写入 `progress.json`。

## 说明

`progress.json` 是本地进度文件，可以删除或通过应用内的重置按钮清空学习状态。项目当前没有额外的配置文件或构建脚本。如果`progress.json`缺失，项目会自动生成新的进度文件。
